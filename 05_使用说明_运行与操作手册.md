# 中国象棋项目使用说明、运行与操作手册

这份文档只讲“怎么运行、怎么操作、文件在哪里看”。如果答辩时老师让你现场演示，优先看这一份。

---

## 一、运行环境

当前工程面向 Windows 开发和运行。

推荐环境：

- Windows 10 / Windows 11
- Visual Studio 2022
- MSVC v143
- EasyX 已安装

重要说明：

- `.exe` 是 Windows 可执行文件，MacBook 不能直接双击运行。
- 如果要在 MacBook 上运行，需要 Windows 虚拟机、双系统、远程 Windows 电脑，或者把项目移植到跨平台图形库。
- 本项目图形界面使用 EasyX，所以正常演示建议在 Windows 上进行。

---

## 二、应该运行哪个程序

当前仓库里能看到两个常见可执行文件：

- `D:\visualstudio\中国象棋\x64\Release\中国象棋.exe`
- `D:\visualstudio\中国象棋\x64\Debug\中国象棋.exe`

答辩演示优先使用：

- `D:\visualstudio\中国象棋\x64\Release\中国象棋.exe`

原因是 Release 版本通常运行更快，尤其是 AI 思考和搜索时更明显。

如果刚改过代码，最稳妥的方式是在 Visual Studio 里打开：

- `D:\visualstudio\中国象棋\中国象棋.sln`

然后选择 `x64` + `Release`，重新生成解决方案。生成后再运行 Release 目录下的 `中国象棋.exe`。

## 三、启动后的主菜单

程序启动后会先出现一个启动器菜单。这个窗口既能进入控制台字符界面对战，也能进入 EasyX 图形界面、局域网联机、观战和重连。

菜单含义如下：

- `1. Local console game`
  本地控制台字符界面对战，使用文本命令走棋，适合展示课程要求中的字符画棋盘和输入解析。
- `2. Local two-player EasyX game`
  本地双人图形对战，两个人在同一台电脑上轮流点击走棋。
- `3. Human vs AI EasyX game`
  人机图形对战，默认红方是玩家，黑方是电脑。
- `4. LAN host EasyX game`
  局域网联机主机，主机执红。
- `5. Join LAN EasyX game`
  显示局域网房间列表并加入对局，客户端执黑；没有发现房间时也可以手动输入 IP。
- `6. Watch LAN game`
  以只读方式观战局域网对局，程序会接收主机同步的当前局面。
- `7. Reconnect LAN client`
  黑方客户端断线后重新连接主机，并同步当前局面继续对战。
- `8. Open PGN replay in EasyX`
  打开 `.pgn` 棋谱文件，并用 EasyX 回放。
- `9. Show leaderboard`
  查看排行榜 CSV 内容。
- `10. Run engine self-tests`
  运行规则、存档、PGN、AI 等自测。
- `11. Exit`
  退出程序。

## 四、开始一局控制台或图形本地对战

操作步骤：

1. 运行 `中国象棋.exe`。
2. 在启动器里输入 `1`。
3. 选择棋盘模式。
4. 输入每步限时，直接回车就是默认 `60` 秒。
5. 选择是否允许悔棋，直接回车就是允许。
6. 选择是否显示合法走法提示，直接回车就是显示。
7. 输入红方和黑方玩家名字，可以直接回车使用默认名。
8. 进入控制台棋盘后，用 `a0 a1` 这类坐标命令走棋，也可以输入 `help` 查看命令。

如果要演示鼠标图形界面对战，在启动器里输入 `2`，后续设置流程相同，进入 EasyX 棋盘窗口后用鼠标点击棋子走棋。

棋盘模式说明：

- `1` 表示标准 `9x10` 中国象棋棋盘。
- `2` 表示扩展 `11x10` 特殊模式。
- 扩展模式只是棋盘横向变宽，棋子走法不变。

## 五、EasyX 棋盘窗口怎么操作

### 1. 走棋

走棋方式很直观：

1. 点击当前方自己的棋子。
2. 程序会高亮显示这个棋子的合法落点。
3. 再点击目标位置。
4. 如果走法合法，棋子会移动，回合切换给另一方。

如果点击了非法位置，程序不会落子，会在右侧状态区域提示原因。

### 2. 棋子显示

棋盘上的棋子使用汉字显示：

- 红方：帅、仕、相、马、车、炮、兵
- 黑方：将、士、象、马、车、炮、卒

这只是显示方式，底层规则仍然由 `Board` 和 `ChessPiece` 判断。

### 3. 右侧按钮

EasyX 对局界面右侧有这些按钮：

- `Undo`
  悔棋。本地双人模式回退一步；人机模式通常回退玩家和 AI 的一整轮。
- `Hint`
  智能提示。程序会计算推荐走法，并高亮“要移动的棋子”和“目标位置”。
- `Save`
  保存当前局面，同时导出当前棋谱。
- `Replay`
  从开局开始自动回放当前这盘棋。
- `Restart`
  重新开始一局。
- `Menu`
  返回 EasyX 内部菜单。
- `Leave LAN`
  联机模式下离开对局。
- `Exit`
  退出当前 EasyX 窗口。

### 4. 胜负提示

当一方胜出、超时、认输或出现和棋时，棋盘中央会出现明显的结果动画和大字提示，例如 `RED WIN`、`BLACK WIN`、`DRAW`。

## 六、人机对战怎么用

操作步骤：

1. 启动程序。
2. 在启动器输入 `2`。
3. 选择棋盘模式和参数。
4. 进入 EasyX 界面后，玩家执红先走。
5. 玩家走完后，右侧状态会显示 `AI thinking...`。
6. AI 算完后会自动落子。

当前 AI 默认档位是 `4`，内部按时间预算思考，不是简单只算一步。它和 `Hint` 共用同一套 `SearchEngine`，所以同一局面下，提示和 AI 实战会尽量保持一致。

人机模式的悔棋逻辑：

- 如果 AI 正在思考时点 `Undo`，会取消 AI 思考，并撤销玩家刚才那一步。
- 如果 AI 已经走完后点 `Undo`，会撤销玩家和 AI 的一整轮。
- AI 思考时，黑方时间会继续减少，这样界面上更直观。

## 七、局域网联机怎么用

当前联机以一名红方主机和一名黑方客户端为对战主体，同时支持房间发现、只读观战和黑方断线重连。

### 1. 两台电脑局域网联机

主机电脑操作：

1. 运行程序。
2. 输入 `4`，选择 `LAN host EasyX game`。
3. 设置棋盘模式、计时、悔棋等参数。
4. 输入端口，直接回车使用默认端口。
5. 等待客户端连接。

客户端电脑操作：

1. 运行程序。
2. 输入 `5`，选择 `Join LAN EasyX game`。
3. 程序会先搜索局域网房间，发现主机后可以按编号加入。
4. 如果没有发现房间，可以选择手动输入主机 IP 和端口。
5. 连接成功后进入 EasyX 对局。

主机 IP 可以在主机电脑的命令行里输入：

```powershell
ipconfig
```

然后查看无线网卡或以太网网卡下面的 `IPv4 地址`。

### 2. 同一台电脑开两个程序测试联机

可以，同一台电脑能开两个。

操作方式：

1. 第一个程序输入 `4` 当主机。
2. 端口用默认值即可。
3. 第二个程序输入 `5` 当客户端。
4. 如果房间列表发现本机房间，直接按编号加入；否则手动输入 `127.0.0.1`。
5. 端口输入和主机相同的端口。

### 3. 观战和断线重连

观战：

1. 主机和客户端开始对局后，主机会继续广播可观战房间。
2. 旁观者在启动器输入 `6`，选择 `Watch LAN game`。
3. 程序会列出可观战房间，选择后进入控制台只读观战。
4. 观战端不能走棋，只接收主机同步的 `STATE` 局面快照。

断线重连：

1. 如果黑方客户端意外断开，主机会提示等待重连。
2. 黑方重新启动程序，输入 `7`，选择 `Reconnect LAN client`。
3. 程序会查找允许重连的房间，也可以手动输入主机 IP 和端口。
4. 重连成功后，主机会发送当前局面快照，客户端恢复到同一盘棋继续对战。

### 4. 联机模式说明

当前工程的联机范围是课程项目级别的局域网直连：

- 主机执红，客户端执黑。
- 主机负责验证走法，是权威状态源。
- 支持正常走棋、悔棋同步、认输、超时和结束同步。
- 支持 UDP 房间发现、只读观战和黑方断线重连。
- 不包含跨公网大厅、账号系统或自动匹配。
- 如果 Windows 防火墙弹窗，需要允许程序在局域网通信。

## 八、存档、棋谱和排行榜在哪里

代码里统一用：

```cpp
std::filesystem::current_path() / "data"
```

作为数据目录。也就是说，数据文件保存在哪里，取决于你从哪个工作目录启动程序。

数据文件分三类：

- 存档文件：`data\saves\*.xqsave`
- 棋谱文件：`D:\visualstudio\中国象棋\pgn\*.pgn`
- 排行榜：`data\leaderboard.csv`

EasyX 里的 `Save` 按钮默认会保存：

- `data\saves\easyx_autosave.xqsave`
- `D:\visualstudio\中国象棋\pgn\easyx_autosave_时间戳.pgn`

棋局结束时，程序也会尝试自动保存当前局面和棋谱，并追加排行榜。PGN 文件使用时间自动命名，所以多盘棋不会互相覆盖。

## 九、PGN 棋谱怎么导出和导入

### 1. 导出 PGN

点击 EasyX 右侧 `Save` 后，程序会导出 `.pgn` 文件。

固定路径是：

- `D:\visualstudio\中国象棋\pgn`

文件名会自动带时间，例如：

- `easyx_autosave_20260521_163012_125.pgn`
- `manual_replay_20260521_163145_009.pgn`

联机时 PGN 只由主机导出。客户端加入别人对局时可以保存本地 `.xqsave`，但不会另外导出 PGN，避免同一台电脑双开测试时两个进程写出重复棋谱。

标准棋盘和扩展棋盘的记谱方式不同：

- 标准 `9x10`：使用 WXF 风格着法。
- 扩展 `11x10`：使用 ICCS 风格坐标着法。

这样做是因为 `11x10` 不是传统中国象棋标准棋盘，如果强行使用中文列编号会产生歧义。

### 2. 导入 PGN 回放

有两个入口：

- 启动器菜单输入 `8`
- EasyX 菜单中点击 `Open PGN Replay`

选择 `.pgn` 文件后，程序会：

1. 读取 PGN 头信息。
2. 根据 `BoardMode` 判断是标准棋盘还是扩展棋盘。
3. 根据 `Format` 判断用 WXF 还是 ICCS 解析。
4. 从初始局面开始逐步复原每一步。
5. 进入只读回放模式。

导入后的棋谱是只读的：

- 不能继续走棋。
- 不能悔棋。
- 不能保存成新对局。
- 可以点击 `Replay` 再次从头播放。

## 十、排行榜怎么查看

排行榜文件是：

- `data\leaderboard.csv`

查看方式有两种：

- 在启动器输入 `6`，直接打印排行榜内容。
- 用 Excel、记事本或 VS Code 打开 `leaderboard.csv`。

排行榜记录内容包括：

- 时间戳
- 棋盘模式
- 红方姓名
- 黑方姓名
- 结果
- 步数
- 双方用时

## 十一、自测怎么运行

有三种方式：

- 启动器输入 `7`
- 命令行运行 `中国象棋.exe --selftest`
- 命令行运行 `中国象棋.exe --smoke`

示例：

```powershell
cd D:\visualstudio\中国象棋\x64\Release
.\中国象棋.exe --selftest
.\中国象棋.exe --smoke
```

`--selftest` 会跑比较完整的规则和功能自测。`--smoke` 是快速冒烟测试，主要确认开局走法和 AI 回复没有崩溃。

## 十二、常见问题

### 1. 为什么启动时还有一个黑色窗口？

这是启动器菜单窗口，用来选择模式。进入 EasyX 对局后，程序会隐藏这个窗口，主要操作都在图形界面里完成。

### 2. 为什么找不到刚保存的文件？

先确认程序是从哪个目录启动的。数据目录永远是“当前工作目录下面的 `data`”。

优先检查：

- `D:\visualstudio\中国象棋\data`
- `D:\visualstudio\中国象棋\x64\Release\data`
- `D:\visualstudio\中国象棋\x64\Debug\data`

### 3. 为什么 MacBook 不能运行这个 exe？

因为 `.exe` 是 Windows 程序，而且项目用了 EasyX。EasyX 也是 Windows 图形库。

### 4. 为什么联机连不上？

优先检查：

- 两个程序是否在同一局域网。
- 客户端输入的 IPv4 地址是否正确。
- 端口是否一致。
- Windows 防火墙是否拦截。
- 如果同一台电脑测试，地址应使用 `127.0.0.1`。

### 5. 为什么导入别人的 PGN 失败？

当前导入器主要支持本项目导出的中国象棋 PGN：

- 标准棋盘使用 WXF。
- 扩展棋盘使用 ICCS 坐标。
- PGN 头里的 `Format` 和 `BoardMode` 最好保留。

如果外部 PGN 使用了别的中文象棋记谱变体，可能需要额外适配。

## 十三、答辩现场推荐演示顺序

推荐按这个顺序演示，最稳：

1. 启动程序，选择 `1`，演示本地 EasyX 双人走棋。
2. 点击一个棋子，展示合法落点高亮。
3. 点击 `Hint`，展示推荐棋子和目标位置高亮。
4. 点击 `Save`，说明生成 `.xqsave` 和 `.pgn`。
5. 回到菜单，选择 `8`，打开刚才导出的 PGN 回放。
6. 重新启动或回菜单，选择 `2`，演示人机对战。
7. 如果时间够，再同一台电脑开两个程序，用 `127.0.0.1` 演示局域网联机。

这样演示能覆盖图形界面、规则判断、提示、存档、PGN 导入回放、人机和联机几个核心亮点。
